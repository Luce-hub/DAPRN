# DAPRN
Dual-stage Attention Prototype Reconstruction Network

## Environment Configuration

The experiments were conducted under the following environment:

* Operating System: Windows 10 (64-bit)
* Python: 3.10
* PyTorch: 2.0.1
* NumPy: 1.24.3
* Pandas: 2.0.3
* Scikit-learn: 1.3.0
* Matplotlib: 3.7.2
* OpenPyXL: 3.1.2

Hardware Configuration:

* CPU: Intel Core i5-12400
* GPU: Intel UHD Graphics 730
* Memory: 16 GB RAM

The required dependencies can be installed using:

pip install -r requirements.txt

The framework consists of three stages:
Meta-training stage
Fine-tuning stage
Testing stage
